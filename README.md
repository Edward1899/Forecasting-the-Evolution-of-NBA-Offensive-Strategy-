# Forecasting the Evolution of NBA Offensive Strategy

This project is based on my MSc Data Science thesis at the University of South Wales.

The analysis explores how NBA offensive strategy has evolved over time, particularly the rise of three-point shooting and changes in offensive efficiency.

Using season-level NBA data, the project applies time-series forecasting models including:

- ARIMA
- SARIMAX
- Prophet

The analysis also includes a simplified efficiency frontier framework for evaluating shot selection.
